# 《闪韵灵镜》谱面同步助手
“谱面同步助手”是基于[**闪韵灵镜 谱面编辑器**](https://cipher-editor-cn.picovr.com/)进行开发的功能拓展插件

旨在给作谱者带来便捷

如有侵犯官方权益，请联系本人进行删除

## 功能介绍
修改谱面编辑器的“下载”按钮，为其增加“同步到VR一体机”的功能。

## 安装说明
1. 前往[**下载页面**](https://github.com/CMoyuer/CipherBeatmapSync/releases)，下载最新的crx和apk文件
